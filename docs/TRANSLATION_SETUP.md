# 번역 기능 설정 가이드

커뮤니티 글 번역 기능을 사용하기 위한 설정 가이드입니다.

## DeepL API 키 발급

1. [DeepL API 페이지](https://www.deepl.com/pro-api?cta=header-pro-api)에 접속
2. 무료 계정 생성 또는 로그인
3. API 키 발급 (무료 플랜: 월 500,000자까지 무료)

## 환경 변수 설정

프로젝트 루트에 `.env.local` 파일을 생성하고 다음 내용을 추가하세요:

```env
NEXT_PUBLIC_DEEPL_API_KEY=your_deepl_api_key_here
```

`your_deepl_api_key_here` 부분을 발급받은 DeepL API 키로 교체하세요.

## 사용 방법

1. 커뮤니티 글 상세 페이지에서 우측 상단의 "번역하기" 버튼 클릭
2. 현재 설정된 언어로 자동 번역됩니다
3. "원문 보기" 버튼으로 원본 텍스트로 돌아갈 수 있습니다

## 지원 언어

- 한국어 (KR) - 원본 언어
- 영어 (EN)
- 일본어 (JP)
- 중국어 (CN)

## 주의사항

- API 키가 설정되지 않은 경우, 번역 기능은 작동하지 않으며 원본 텍스트가 표시됩니다
- 무료 플랜의 경우 월 500,000자까지 번역 가능합니다
- 번역은 실시간으로 DeepL API를 호출하여 수행됩니다
