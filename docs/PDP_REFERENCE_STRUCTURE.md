# PDP 페이지 레퍼런스 구조

## 시술 PDP 레퍼런스 구조
1. 상단 배너 (이미지 + 시술명 + 할인율)
2. 시술명 및 평점 (별점, 리뷰 수, Q&A 수)
3. 옵션 선택 (현재 옵션 표시)
4. 가격 정보
5. 병원사례 (Before/After 이미지 그리드) - 추가 필요
6. 리뷰 (별점, 리뷰 수만)
7. 병원 정보 (병원명, 평점, 후기 수, 시술 수)
8. 시술 설명 - 추가 필요
9. 옵션 선택 (체크박스 + 옵션 목록)
10. 이런 분에게 추천해요 - 추가 필요
11. 시술 과정 (단계별) - 추가 필요
12. 다운타임 - 추가 필요
13. 부작용 안내 - 추가 필요
14. 의료진 프로필 - 추가 필요
15. 위치 및 오시는 길 (지도, 주소) - 추가 필요
16. 병원 사진 - 추가 필요
17. Q&A 섹션 - 추가 필요
18. 관련 이벤트 - 추가 필요
19. 하단 버튼 (찜하기, 예약하기)

## 병원 PDP 레퍼런스 구조

9. 환자 만족도 통계 - 추가 필요
10. 위치 상세 (지도)

14. 소속 의사 목록
