# BeauTrip

뷰티 여행을 위한 Next.js 기반 웹 애플리케이션

## 시작하기

### 설치

```bash
npm install
```

### 개발 서버 실행

```bash
npm run dev
```

브라우저에서 [http://localhost:3000](http://localhost:3000)을 열어 확인하세요.

## 주요 기능

- 🏠 홈 화면
- 🔍 검색 기능
- 📍 지역 선택
- 🏷️ 필터 태그
- 📱 반응형 디자인
- 🎨 민트톤 디자인 시스템

## 기술 스택

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
- React Icons

## 색상 팔레트

- Primary Light: `#37EAD0`
- Primary Main: `#3ED4BE`
